**2. Create `src/lib/services/transcription.service.ts`:**

Follow the same patterns as recording.service.ts (Result type, captureError helper, Sentry integration).

Export `TranscriptionResult<T>` type (same discriminated union pattern as RecordingResult).

Export `async function transcribeRecording(recordingSessionId: string): Promise<TranscriptionResult<{ segmentCount: number }>>`:

1. Find RecordingSession by ID. Return error if not found or no audioStoragePath.
2. Update session status to PROCESSING.
3. Get signed URL via `getRecordingSignedUrl(session.audioStoragePath)`.
4. Fetch audio from signed URL, convert to ArrayBuffer, create File object with name `{recordingSessionId}.webm` and type `audio/webm`.
5. Call OpenAI transcription:
   ```typescript
   const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });
   const transcription = await openai.audio.transcriptions.create({
     model: 'gpt-4o-transcribe-diarize',
     file: audioFile,
     language: 'ja',
     response_format: 'diarized_json',
     chunking_strategy: 'auto',
   });
   ```
6. Parse response: The diarized_json response contains speaker segments. Map each to a TranscriptionSegment record:
   - `recordingId`: recordingSessionId
   - `segmentIndex`: index in array
   - `speakerLabel`: segment.speaker or null
   - `content`: segment.text
   - `startMs`: Math.round(segment.start * 1000)
   - `endMs`: Math.round(segment.end * 1000)
   - `language`: 'ja'
7. Use `prisma.transcriptionSegment.createMany({ data: segments })`.
8. Update session status to COMPLETED.
9. Return `{ success: true, data: { segmentCount: segments.length } }`.

On any error:
- Update session status to FAILED (best-effort, wrap in try/catch).
- Call captureError helper (Sentry + console.error).
- Return `{ success: false, error: message }`.

**Important:** The OpenAI SDK types for diarized_json may not be fully typed. Use type assertions where needed and add a comment noting the response format. Inspect the actual response shape — it may be `transcription.speakers` or `transcription.segments` or a similar field. Handle both possibilities defensively.

Create the OpenAI client lazily (not at module level) so the module can be imported even without OPENAI_API_KEY set (important for tests and build).

- Check admin auth via `getAdminSession()`. Return 401 if not admin.
- Parse JSON body, extract `recordingSessionId`. Return 400 if missing.
- Call `transcribeRecording(recordingSessionId)`.
- If success, return 200 with `{ success: true, segmentCount }`.
