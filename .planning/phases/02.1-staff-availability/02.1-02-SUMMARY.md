## Decisions Made

- **generateTimeSlots for time options**: `generateTimeSlots('06:00', '23:30', 30)` produces `['06:00', '06:30', ..., '23:00']` â€” the last slot `23:00` fits because 23:00 + 30min = 23:30 which equals the end bound. Provides full business-hours coverage.
- **Hidden input for toggle state**: Unchecked HTML checkboxes are not submitted in FormData. Using `<input type="hidden" name="day_N_isAvailable" value="true|false" />` ensures the current toggle state is always submitted, matching what the server action expects.
- **schedule key in workersPage namespace**: Adding `"schedule"` to `admin.workersPage` (not `admin.schedulePage`) avoids a second `useTranslations` call in `WorkerTable`, keeping the component clean.
