**Create `src/lib/services/reporting.service.ts`**:

Revenue aggregation functions that query the Booking table (status = 'CONFIRMED') joined with Service (for price):

1. `getRevenueSummary(params: { startDate: Date; endDate: Date; groupBy: 'day' | 'week' | 'month' })`:
   Returns array of `{ period: string; totalRevenue: number; bookingCount: number; newCustomerCount: number; existingCustomerCount: number }`.

   Implementation:
   - Query bookings where startsAt between startDate and endDate AND status = 'CONFIRMED'
   - Join Service to get price
   - Group by date (truncated to day/week/month using Prisma raw query or groupBy)
   - For new vs existing customer: a customer is "new" if this was their first-ever booking (check if they have no CONFIRMED bookings before startDate)
   - Total revenue = SUM(service.price) for confirmed bookings in each period

2. `getWorkerMetrics(params: { startDate: Date; endDate: Date })`:
   Returns array of `{ workerId: string; workerName: string; totalRevenue: number; bookingCount: number; averagePerBooking: number }`.

   Implementation:
   - Query bookings in date range, status = 'CONFIRMED'
   - Group by workerId
   - Join Worker for name, Service for price
   - Calculate: totalRevenue = SUM(service.price), bookingCount = COUNT(*), averagePerBooking = totalRevenue / bookingCount
   - Sort by totalRevenue DESC

3. `getDashboardTotals(params: { startDate: Date; endDate: Date })`:
   Returns `{ totalRevenue: number; totalBookings: number; uniqueCustomers: number; averageRevenuePerBooking: number }`.

For the date grouping, use Prisma `$queryRaw` with PostgreSQL `date_trunc()` for grouping by day/week/month. This is more efficient than fetching all bookings and grouping in JavaScript.

Example raw query for daily revenue:
```sql
SELECT date_trunc('day', "startsAt") as period,
       SUM(s.price) as total_revenue,
       COUNT(*) as booking_count
FROM "Booking" b
JOIN "Service" s ON b."serviceId" = s.id
WHERE b."startsAt" >= $1 AND b."startsAt" < $2
  AND b.status = 'CONFIRMED'
GROUP BY period
ORDER BY period ASC
```
