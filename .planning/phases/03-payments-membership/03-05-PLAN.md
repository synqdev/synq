  <name>Task 1: Create QR service and API routes</name>
  <files>
    src/lib/services/qr.service.ts

    app/api/membership/verify/route.ts

  <action>
    1. Create `src/lib/services/qr.service.ts`:

       **`generateMembershipQR(customerId: string): Promise<string>`**
       - Use jose (already installed) `SignJWT` to create a JWT:
         - `sub`: customerId
         - `type`: 'membership'
         - Protected header: `{ alg: 'HS256' }`
         - Set issuedAt and expirationTime to '24h'
         - Sign with `QR_MEMBERSHIP_SECRET` env var (fallback to 'dev-qr-secret' in dev)
       - Use `qrcode` package (`QRCode.toDataURL(token, { errorCorrectionLevel: 'M', width: 300, margin: 2 })`)
       - Return the data URL string (data:image/png;base64,...)

       **`verifyMembershipQR(token: string): Promise<{ customerId: string } | null>`**
       - Use jose `jwtVerify` to verify token against same secret
       - Check `payload.type === 'membership'` and `typeof payload.sub === 'string'`
       - Return `{ customerId: payload.sub }` on success
       - Return `null` on any failure (expired, invalid, wrong type)

       Secret key: `const QR_SECRET = new TextEncoder().encode(process.env.QR_MEMBERSHIP_SECRET ?? 'dev-qr-secret')`

    2. Create `app/api/membership/qr/route.ts`:

       **GET handler**: Generate QR code for authenticated customer
       - Read `customerId` from cookies
       - If no cookie, return 401
       - Check customer has active membership via `getActiveMembership(customerId)` from membership service
       - If no active membership, return 403 with message "No active membership"
       - Call `generateMembershipQR(customerId)` to get data URL
       - Return `{ qrDataUrl, expiresIn: '24h', membershipStatus: membership.status, planName: membership.plan.name }`

    3. Create `app/api/membership/verify/route.ts`:

       **POST handler**: Verify scanned QR token (staff use)
       - Verify admin session — return 401 if not authenticated (only staff can verify)
       - Parse request body: `{ token: string }`
       - Call `verifyMembershipQR(token)` from QR service
       - If null (invalid/expired): return `{ valid: false, error: 'Invalid or expired QR code' }`
       - If valid: look up Customer by customerId, check active membership
       - Return `{ valid: true, customer: { name, email, membershipStatus, planName, currentPeriodEnd } }`
       - Include visit tracking: increment `Customer.visitCount` and set `Customer.lastVisitDate` to now
  </action>
  <verify>
    - `npx tsc --noEmit` passes
    - QR service exports generateMembershipQR and verifyMembershipQR
    - QR route returns data URL for authenticated customer with active membership
    - Verify route checks admin session and returns customer membership details
  </verify>
  <done>QR service generates signed 24h JWT tokens encoded as QR data URLs. GET /api/membership/qr returns QR for customer. POST /api/membership/verify validates scanned token and returns membership status with visit tracking.</done>
