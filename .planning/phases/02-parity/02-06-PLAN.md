**Extend `src/lib/services/reporting.service.ts`** with:

1. `getWorkerRankings(params: { startDate: Date; endDate: Date })`:
   Returns array of:
   ```typescript
   {
     rank: number;
     workerId: string;
     workerName: string;
     totalRevenue: number;
     bookingCount: number;
     differenceFromFirst: number;  // revenue difference from rank 1
   }
   ```
   Implementation:
   - Same as getWorkerMetrics but adds: rank (1-based index after sorting by revenue DESC) and differenceFromFirst (rank1.totalRevenue - current.totalRevenue).

2. `getRepeatCustomerRate(params: { startDate: Date; endDate: Date })`:
   Returns:
   ```typescript
   {
     totalCustomers: number;        // unique customers with bookings in range
     repeatCustomers: number;       // customers with >1 CONFIRMED booking ever
     repeatRate: number;            // percentage (0-100, 1 decimal)
     newCustomers: number;          // first-ever booking was in this range
   }
   ```
   Implementation:
   - Get all unique customerIds with CONFIRMED bookings in the date range
   - For each, check if they have any CONFIRMED booking BEFORE startDate
   - If yes: repeat customer. If no: new customer.
   - repeatRate = (repeatCustomers / totalCustomers) * 100
   - Use a raw query or two-step process for efficiency:
     Step 1: Get unique customerIds with bookings in range
     Step 2: Count how many of those have bookings before startDate
